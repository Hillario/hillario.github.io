# REACT CRUD APP USING REACT HOOKS AND REACT CONTEXT API
React app with CRUD operations implemented with react hooks and the context api which provides a way to pass data through the component tree without having to pass props down manually at every level.

## Available Scripts

In the Context Folder File, the GlobalState.js file elucidates how to achieve CRUD operations on an external file in this case, a file accessed by the context api. The process is as follows: 

### `Step 1`

Create an initial state

### `Step 2`

Create the context which is an instance of the GlobalContext class

### `Step 3`

Create the provider component 

### `Step 4`

Define Actions(CRUD OPERATIONS)
